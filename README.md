# bs-react-native-paper

[React Native Paper bindings](https://github.com/callstack/react-native-paper) for BS

```ocaml
<Paper.Button onPress=((_event) => yourAction())>
  <Paper.Text>
    (ReasonReact.string("Click me"))
  </Paper.Text>
</Paper.Button>
```
